# Generate CEO Briefing

You are tasked with generating a comprehensive Weekly CEO Briefing for Usama's Personal AI Employee system.

## Your Task

Analyze the week's activities and generate a strategic briefing that includes:

### 1. Executive Summary
- High-level overview of the week
- Key achievements
- Overall progress status

### 2. Goals Progress Analysis
- Read Business_Goals.md
- Check progress toward each goal
- Identify which goals are on track vs. behind schedule

### 3. Completed Tasks Review
- Scan the Done folder for all completed tasks
- Categorize tasks by type (LinkedIn, planning, development, etc.)
- Count total tasks completed this week

### 4. Bottleneck Identification
- Identify tasks that took longer than expected
- Note any recurring issues or challenges
- Highlight areas needing attention

### 5. Proactive Suggestions
- Recommend next priorities based on Business_Goals.md
- Suggest improvements or optimizations
- Flag any risks or concerns

### 6. Upcoming Focus Areas
- What should be prioritized next week?
- Any approaching deadlines?
- Recommended action items

## Steps to Complete

1. **Read Business_Goals.md** from the vault
   - Understand current objectives and targets
   - Note success criteria

2. **Analyze Done folder**
   - Read all completed task files
   - Look for patterns and insights
   - Calculate completion velocity

3. **Review Dashboard.md and logs**
   - Check system status
   - Review recent AI Employee actions
   - Note any errors or issues

4. **Generate Briefing File**
   - Create file in Briefings folder
   - Name format: `CEO_Briefing_YYYY-MM-DD.md`
   - Use the template structure below

5. **Update Dashboard.md**
   - Add link to the new briefing
   - Update status if needed

## Important Guidelines

- Be data-driven: Use actual numbers from files
- Be honest: Don't sugarcoat issues or bottlenecks
- Be actionable: Every suggestion should be specific
- Be forward-looking: Focus on what's next
- Be concise: Respect the CEO's time (2-3 pages max)

## Success Criteria

A great briefing should:
- Take 5 minutes to read
- Give complete picture of the week
- Enable quick decision-making
- Highlight both wins and issues
- Provide clear next steps

## After Generation

1. Save briefing in `/Briefings/` folder
2. Add entry to Dashboard.md under "Recent Briefings"
3. Output message: "CEO Briefing generated: [filename]"
