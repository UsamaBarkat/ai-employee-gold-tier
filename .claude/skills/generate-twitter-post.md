# Generate Twitter Post

You are tasked with generating professional Twitter/X posts for Usama's AI Employee system.

## Your Task

Create engaging, professional Twitter posts to share progress, learnings, and achievements.

## Content Guidelines

### Character Limit
- **Maximum: 280 characters** (including hashtags and spaces)
- Aim for 240-260 for comfortable reading
- Use emojis sparingly (1-3 max)

### Hashtag Strategy
- Use 3-5 relevant hashtags
- Required: `#AgenticAI` `#Panaversity`
- Additional: `#AI` `#Python` `#BuildInPublic` `#Hackathon` `#ClaudeCode`

### Tone & Style
- Professional but friendly
- Share genuine learnings
- Celebrate progress
- Be authentic
- Focus on value

## Post Types

### 1. Progress Update
Share what you've accomplished:
```
Completed [Feature Name]! 🎉

✅ [Key achievement 1]
✅ [Key achievement 2]
✅ [Key achievement 3]
```

### 2. Learning Share
Share insights:

Share technical details:
```

```

## Generation Process

1. **Identify the Topic**
   - What do you want to share?
   - Why is it interesting/valuable?

2. **Draft the Content**
   - Write main message (1-3 sentences)
   - Add relevant details
   - Keep it under 280 chars

3. **Add Hashtags**
   - Include required: `#AgenticAI` `#Panaversity`
   - Add 1-3 more relevant tags

4. **Add Emojis** (Optional)
   - Use 1-3 relevant emojis
   - Place strategically (beginning or bullet points)

5. **Review**
   - Check character count
   - Verify clarity
   - Ensure professional tone

Create a file in `/Needs_Action/` with this structure:
